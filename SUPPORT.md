# Support Policy

Use GitHub Issues for reproducible defects and feature proposals, and GitHub Discussions for usage questions and design conversation. Security reports follow `SECURITY.md`.

Community support is best-effort and carries no response-time SLA. Include the Framekit package versions, Node.js version, deployment adapter, minimal reproduction, expected behavior, and relevant sanitized logs. The compatibility matrix in `docs/compatibility.md` defines the environments maintainers test; unsupported combinations may be closed unless a contributor can provide a fix and verification.

The latest published `0.x` minor is the supported pre-1.0 line. Upgrade assistance, custom architecture work, and operational incident response are outside the community support commitment.
